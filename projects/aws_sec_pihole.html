Serverless Ad Blocking with Cloudflare Gateway

Description

    2023 - ongoing blog source code

Terraform module that automates the setup needed to mimic the Pi-hole's behaviour using only serverless technologies (Cloudflare Gateway, to be precise), as described in Serverless Ad Blocking with Cloudflare Gateway.

In short, this module can be used to create:

  1. A set of Cloudflare Lists which contain the list of domains to block.
  2. A Cloudflare Gateway Policy which blocks access (at the DNS level) to those domains.
